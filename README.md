## GlideSample    

### app模块说明   
1、使用Glide加载圆形或者圆角的图片，其中圆角的弧度可以自定义   

### 防盗链   
1、利用referer做防盗链，在浏览器访问站内图片的时候，在图片本站是可以看得，在外头就看不到
